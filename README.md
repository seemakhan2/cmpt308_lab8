# cmpt308_lab8 # 

Lab 8 - Roles, Functions, and Triggers

# How to Run

1. Open pgAdmin
2. Select a PostgreSQL database
3. Open the Query Tool
4. Run the SQL in `cmpt308_lab8.sql`
5. Copy the query results into the required PDF.
6. Read comments.

# What This Lab Demonstrates

This lab demonstrates database roles and privileges, GRANT and REVOKE, a stored function for course registration, and a trigger that automatically records enrollment events in an audit table.
